# Nuxt3 + UnoCSS 重构的 [hackernews](https://github.com/nuxt/hackernews) 博客网站

## 启动

安装依赖:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```

## 开发服务


```bash
pnpm dev
```
访问 `http://localhost:9999`

## 生产


打包:
```bash
pnpm build
pnpm generate
```

预览:

```bash
pnpm preview
```
